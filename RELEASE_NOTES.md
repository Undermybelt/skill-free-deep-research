First release of free-deep-research skill.

## Features
- Research pipeline: Plan → Execute → Verify → Synthesize
- Multi-source: web_search, github, arxiv (configurable)
- Parallel execution with retry logic
- Markdown reports saved to memory/research/

## Usage
```
/research start --topic "AI agent orchestration" --depth medium
/research status <taskId>
```
